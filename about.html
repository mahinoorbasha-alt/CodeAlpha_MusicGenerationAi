About the Project

Overview

Music Generation with AI is a full-stack Flask application that teaches a Long Short-Term Memory (LSTM) neural network to compose music by learning sequential patterns from MIDI files. It was built as part of the CodeAlpha Machine Learning internship.

How it works

  1. Upload — MIDI files are stored in the dataset/ directory.
  2. Preprocess — Music21 parses each file and extracts notes and chords.
  3. Train — Two stacked LSTM layers (256 units) learn the next-note distribution.
  4. Generate — A random seed sequence is fed back into the model to compose a new track.
  5. Download — The new sequence is written to a `.mid` file with Music21.

Tech Stack

Python · Flask · TensorFlow / Keras · Music21 · NumPy · HTML / CSS / JavaScript
